# 1.1.1

## 変更点

- ノッチを無視するための水平方向と垂直方向のオプションをそれぞれ追加しました。
- コミュニティコメントのコピー機能を追加しました。
- サーバー情報とリストにタイトルを追加しました。
- エンジンにセーフエリアを追加しました。
- エンジンのデバッグモードに、パフォーマンス記録データを追加しました。
- iOSにおいて消音モードを無視するように変更しました。
- UI/UXを改善しました。
- パフォーマンスが向上しました。
- Sonolusリンクから開く機能を改善しました。
- 一部のAndroid端末で振動フィードバックが機能しない問題を修正しました。
- 破損した音声ファイルが一部のAndroid端末で画面が暗転させる問題を修正しました。
- 一部の音声ファイルがBGMを二重に再生させる問題を修正しました。
- お気に入りからユーザーを削除できない問題を修正しました。
