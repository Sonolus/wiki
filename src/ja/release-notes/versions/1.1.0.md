# 1.1.0

## 変更点

- カスタム サーバーにユーザー機能を追加しました。
- カスタム サーバーにカスタム情報ボタンを追加しました。
- カスタム サーバーにファイル検証機能を追加しました。
- エンジンに触覚フィードバック機能を追加しました。
- エンジンに進捗グラフを追加しました。
- エンジンにカスタムライフ機能を追加しました。
- エンジンにアーキタイプごと、エンティティごとのスコア機能を追加しました。
- エンジンにアーキタイプごと、エンティティごとのライフ機能を追加しました。
- エンジンへのインポートにデフォルト値を追加しました。
- 設定にステータスアニメーションオプションを追加しました。
- UI/UXを改善しました。
- パフォーマンスが向上しました。
- iOS で一時停止中に発生する遅延を修正しました。
