# 04. 舞台

在本章中，我们将实现舞台原型。

## 生命周期

舞台实体应始终保持活跃，一个简单地实现方法就是在 `spawnTime` 和 `despawnTime` 回调函数中返回一个巨大的值：

::: code-group

```TypeScript
export class Stage extends Archetype {
    spawnTime() {
        return -999999
    }

    despawnTime() {
        return 999999
    }

    // ...
}
```

```JavaScript
export class Stage extends Archetype {
    spawnTime() {
        return -999999
    }

    despawnTime() {
        return 999999
    }

    // ...
}
```

:::

## 绘制

绘制舞台与游玩模式类似：

::: code-group

```TypeScript
export class Stage extends Archetype {
    // ...

    updateParallel() {
        const layout = new Rect({
            l: judgeLine.l,
            r: judgeLine.r,
            t: 1 - note.radius / 4,
            b: 1 + note.radius / 4,
        })

        skin.sprites.judgeLine.draw(layout, 0, 1)
    }
}
```

```JavaScript
export class Stage extends Archetype {

```
