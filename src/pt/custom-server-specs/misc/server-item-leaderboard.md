# `ServerItemLeaderboard`

`ServerItemLeaderboard` fornece informações sobre uma tabela de classificação de itens.

## Sintaxe

```ts
type ServerItemLeaderboard = {
    name: string
    title: Text | (string & {})
    description?: string
}
```

## Exemplos

```json
{
    "name": "...",
    "title": "..."
}
```
