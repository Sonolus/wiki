# 1.1.1

## 변경사항

- 노치 무시에 가로 및 세로로 분리 옵션을 추가했습니다.
- 커뮤니티 댓글에 복사 기능을 추가했습니다.
- 서버 정보 및 목록에 제목을 추가했습니다.
- 엔진에 안전 구역을 추가했습니다.
- 엔진 디버그 모드에 성능 데이터 기록을 추가했습니다.
- iOS의 무음 모드를 무시하게 변경했습니다.
- UI/UX를 개선했습니다.
- 성능을 개선했습니다.
- Sonolus 링크 열기를 개선했습니다.
- 일부 Android 기기에서 햅틱 반응이 작동하지 않는 문제를 수정했습니다.
- 일부 Android 기기에서 잘못된 오디오 파일로 인해 검은 화면이 나오는 문제를 수정했습니다.
- 일부 오디오가 배경음을 두 번 재생하는 문제를 수정했습니다.
- 즐겨찾기에서 사용자를 제거할 수 없는 문제를 수정했습니다.
