# 1.1.1

## Lista de cambios

- Se agregaron opciones horizontales y verticales separadas para ignorar el notch.
- Se añadió la opción de copiar en los comentarios de la comunidad.
- Se añadió un título a la información y listado del servidor.
- Se ha añadido una zona segura al motor.
- Se ha añadido la opción de registrar datos de rendimiento al modo depuración del motor.
- Ahora la aplicación ignorará el modo silencioso en iOS.
- Interfaz de usuario/experiencia de usuario mejorada.
- Rendimiento mejorado.
- Se ha mejorado la apertura de los enlaces de Sonolus.
- Se solucionó un problema por el que la respuesta háptica no funcionaba en algunos dispositivos Android.
- Arreglado un problema que provocaba que archivos de audio corruptos causaran una pantalla negra en algunos dispositivos Android.
- Solucionado un problema en el que algunos archivos de audio hacían que la música se reprodujera doble.
- Arreglado un problema que provocaba que no se pudieran eliminar usuarios de favoritos.
